## 🚀 Cómo usar

1. Cloná o descargá este repositorio
2. Abrí cada archivo `.html` directamente en **Chrome o Safari**
3. No requiere servidor ni instalación — funciona offline

```bash
git clone https://github.com/TU_USUARIO/medical-quimica-visita.git
cd medical-quimica-visita
open Relevamiento_MedicalQuimica.html
open Analisis_Licitaciones_MedicalQuimica.html
```

## 📊 Datos clave del análisis

- **192 participaciones** registradas en DNCP
- **150 adjudicaciones** — tasa de éxito del **78%**
- **42 licitaciones perdidas** — oportunidad de mejora con IA
- Recuperación estimada con IA: **~8 adjudicaciones adicionales/año**
